# Francisco Negrete - Portfolio

Professional portfolio website for Data Analyst & Engineer role.

## Technologies
- React 18+
- Vite
- Tailwind CSS
- Lucide React (Icons)

## Setup and Run

Since the Node.js environment was not detected automatically, please follow these manual steps to run the project:

1. **Install Node.js**: Ensure you have Node.js installed (v18+ recommended).
2. **Install Dependencies**:
   Open a terminal in this folder and run:
   ```bash
   npm install
   ```
3. **Run Development Server**:
   ```bash
   npm run dev
   ```
4. **Build for Production**:
   ```bash
   npm run build
   ```

## Structure
- `src/components`: UI components (Navbar, Hero, About, etc.)
- `src/data`: Content data (content.json)
- `src/styles`: Global styles
- `tailwind.config.js`: Custom color palette and theme configuration.
